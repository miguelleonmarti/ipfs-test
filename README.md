## Getting started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/miguelleonmarti/multi-sig-wallet.git
```

2. Install dependencies

```sh
npm install
```

3. Run the test

```sh
npm run test
```
